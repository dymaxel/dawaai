Odoo 13 POS Combo (Enterprise)

POS Combo module allows user to use combo feature in restaurant.

Note: This module is tested and working in Enterprise Edition. And we are supporting Ubuntu OS and not Windows OS.

Available for Community

How to use

  • We have one features available in POS Configuration as per below image.


  • Create Combo


  • Select combo product, open popup of combo product.




  • Combo product details added to cart.


  • Combo product print on receipt.


  • POS Order with combo product details.
